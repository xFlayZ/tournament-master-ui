Information

Schritt 1 - Spieler hinzufügen

Gebe einen Spielernamen ein und drücke den Button "Spieler hinzufügen"

Schritt 2 - Modus auswählen

Wähle einen Modus "301" oder "501" aus.

Schritt 3 - Schwierigkeit auswählen

Wähle eine Schwierigkeit "Single-Out" oder "Double-Out"

- Single-Out: hierbei muss zum Abschluss bei einem Restwert von Beispielsweise "12" mindestens die "12" getroffen werden

- Double-Out: hierbei muss zum Abschluss bei einem Restwert von Beispielsweise "12" die doppel "6" getroffen werden

Schritt 4 - Spiel starten

Zum Starten auf dem Button "Spiel starten" drücken.

Spieler löschen

Wenn ein Spieler gelöscht werden soll, drücken Sie auf das Symbol mit dem Papierkorb [] hinter dem Spielernamen
